# Interactive Dashboard for Analyzing Renewable Energy Trends Over a Decade

This project provides an interactive dashboard to visualize and analyze the trends in renewable energy adoption in Abu Dhabi from 2013 to 2024. The dashboard uses the dataset "Renewable Energy Share by Year (2013-2024)" and includes features such as trend visualization, year-over-year analysis, predictive analytics, and exportable insights.

## Features
- **Trend Visualization:** Interactive charts for exploring renewable energy growth.
- **Year-over-Year Analysis:** Compare renewable energy shares for specific years.
- **Predictive Analytics:** Forecast future renewable energy shares using AI models.
- **Data Filtering:** Filter data by specific years of interest.
- **Downloadable Insights:** Export visualizations in formats like PNG and PDF.
- **Insights Section:** Contextual analysis for policy and investment decisions.

## Usage

### Requirements
- Python 3.7+
- Required Libraries: `pandas`, `plotly`, `dash`

### Installation
1. Clone the repository:
   bash
   git clone https://github.com/YourUsername/Renewable-Energy-Dashboard.git
   cd Renewable-Energy-Dashboard
   
2. Install the required dependencies:
   bash
   pip install pandas plotly dash
   
3. Run the app:
   bash
   python app.py
   
4. Open your web browser and navigate to `http://127.0.0.1:8050/` to access the dashboard.

### How to Use
1. **Select Years:** Use the dropdown menu to select specific years for analysis.
2. **View Trends:** Explore the interactive graph to see trends in renewable energy adoption.
3. **Export Insights:** Use the download options to save charts and data for your reports or presentations.
4. **Read Insights:** Navigate to the insights section for additional analysis and context.
